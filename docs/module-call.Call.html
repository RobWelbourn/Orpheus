Class: Call

call.Call()

Represents the state of a call. This class wraps a Twilio VoiceResponse object used to generate TwiML. A Call object responds to webhooks and status callbacks, which settle a Promise that allows the next step in the script to be executed.
Call properties are derived from the results of an outbound API call, webhooks or status callbacks, with the exceptions of eventSource and childCalls. The most important ones are listed below. A definitive list is returned by the propertyMappings property. Some properties are only applicable to childCalls.
